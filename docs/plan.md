[# Plan – Menyentuh website](docs/plan.md:1)

[## Doel](docs/plan.md:3)
Een statische, meertalige (Nederlands) website voor de massagepraktijk **Menyentuh** van **Quirina Maas**. De site moet een krachtige landingspagina hebben, extra informatiepagina’s, en een eenvoudige afspraak‑aanvraag met zichtbare tijdslots plus WhatsApp‑contact.

[## Pagina’s](docs/plan.md:7)
- **Home (Landing)**: hero, USP’s, behandelingen, voordelen, afspraak‑sectie met tijdslots, testimonials, CTA’s.
- **Over**: visie, werkwijze, praktijkwaarden, korte bio.
- **Behandelingen**: overzicht van massages + aromatherapie met beschrijvingen, duur en indicaties.
- **Tarieven**: overzicht van prijzen per behandeling, duur en wat inbegrepen is.
- **Contact**: contactgegevens, openingstijden, locatie, WhatsApp‑knop, formulier met afspraak‑aanvraag.

[## Afspraak‑mock](docs/plan.md:15)
- Toon openingstijden en een set tijdslots (statisch).
- Gebruiker kiest tijdslot (UI highlight).
- Formulier voor naam, e‑mail, telefoon, behandeling en bericht.
- Knoppen voor **Aanvraag per e‑mail** (mailto) en **Aanvraag via WhatsApp** (wa.me) met automatisch samengestelde tekst.

[## Visueel ontwerp](docs/plan.md:22)
- **Kleuren**: pastel groen, goud, wit‑pastel als basis; zachte contrastkleur voor tekst.
- **Typografie**: elegante serif voor koppen, rustige sans‑serif voor body.
- **Layout**: ruime witte ruimte, kaarten, zachte schaduwen, afgeronde hoeken.
- **CTA’s**: duidelijke primaire knop (goud) en secundaire knop (outline groen).

[## Structuur](docs/plan.md:29)
```
index.html
over.html
behandelingen.html
tarieven.html
contact.html
css/styles.css
js/main.js
```

[## Implementatiestappen](docs/plan.md:39)
1. Basisskelet HTML per pagina met gedeelde header/footer.
2. CSS ontwerp met kleurpalet, typografie en componenten.
3. JS interactie voor tijdslot‑selectie en bericht‑generatie.
4. Content in volledig Nederlands en consistent met merk.
5. Documentatie bijwerken.
