[# Menyentuh](README.md:1)

Een serene, meertalige landingwebsite in het Nederlands voor massagepraktijk **Menyentuh** van Quirina Maas. De site bevat meerdere pagina’s met informatie over behandelingen, tarieven en contact, plus een statische tijdslot‑weergave en WhatsApp‑knop.

[## Pagina’s](README.md:5)
- Home (landing): `src/index.html`
- Over: `src/over.html`
- Behandelingen: `src/behandelingen.html`
- Tarieven: `src/tarieven.html`
- Contact: `src/contact.html`

[## Starten](README.md:12)
Open `src/index.html` in je browser. Er is geen build‑stap nodig.

[## Aanpassen](README.md:15)
- Kleuren, typografie en layout: `src/css/styles.css`
- Interactie (tijdslot selecteren + mailto): `src/js/main.js`
- Content per pagina: bestanden in `src/`
